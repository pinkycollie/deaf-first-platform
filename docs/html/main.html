Deaf-First Principles

How we
build

  • 01 Visual-first UI — every system state communicable within 2 seconds of first render, without sound.
  • 02 Caption-first video — captions are not an option. They are the default track, always on.
  • 03 ASL priority overlays — sign language is a first-class UI element, not an accessibility afterthought.
  • 04 No audio gates — zero features locked behind audio interaction. Phone verification, audio CAPTCHA — banned.
  • 05 Trust is structural — Fibonrose trust scoring is enforced at protocol level, not bolted on as a policy.
  • 06 Lifetime support — Sunset stage includes permanent accommodation registration that never expires.
